DEMO: Drawing Board

Educational and productive game

This is a semi-finished educational game demo with the theme of drawing called Drawing Board

You can paint without a drawing board. Freely draw lines or patterns with various gestures facing the camera.

Platform | PC
Date | 2021
Tools | Python Gesture Recognition

Operation

Index finger: draw a line | select tool
Pinch the middle finger of the index finger: zoom in and out the shape (graphic)

Purpose of the game demo

1 Education and fun
2 Give full play to artistic talents in the game and make artistic creations
3 Experience novel interactive methods and feel the beauty of painting in the sky

Interface & Interaction

Draw a single line

Draw a single rectangle

Draw multiple shapes

Draw a Peggi
